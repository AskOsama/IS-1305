How Caesar Cipher Works

The Caesar cipher is a substitution cipher where each letter is replaced by a letter a fixed number of positions down the alphabet.

Encryption: C = (P + K) mod 26
Decryption: P = (C - K) mod 26

Where:

  • P = Position of plaintext letter (A=0, B=1, ..., Z=25)
  • K = Shift key (1-25)
  • C = Position of ciphertext letter

Security Note: The Caesar cipher is extremely weak because there are only 25 possible keys. A brute-force attack can try all possibilities in seconds!
